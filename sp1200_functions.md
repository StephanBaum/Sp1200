1. The Sample Module (Red Button)
To enter this mode, press the Sample button. The red LED above it will illuminate.
Option 1: Volume Unit (VU) Mode (Default)
Behavior: Monitors incoming audio levels.
Display: Top row shows the current selected Bank/Pad (e.g., A1) and Input Gain setting (e.g., +20dB). If a pad already contains a sound, an asterisk (*) appears next to the pad number. The bottom row displays a real-time LCD visual VU meter.
Controls: The physical Gain knob on the top right adjusts the input level.
Option 2: Assign Voice / Select Pad
Behavior: Chooses which pad the next sample will be recorded to.
Display: Sampling A1 (or whichever pad is selected). If the selected pad already has a sound, an asterisk appears, and the display asks to confirm erasure.
Controls: Press a pad to select it. Press Yes on the keypad to erase an existing sound, or No to browse for a vacant pad. Press Enter to confirm a vacant pad.
Sub-menu (Output Channel): After confirming a pad, the display asks for an output channel (1-8). Enter the number via the keypad and press Enter. You are then returned to Option 1 (VU Mode).
Option 3: Level (Internal Preamp)
Behavior: Adjusts the internal preamp gain for incoming audio.
Display: Shows Input Gain and the current dB level.
Controls: Use the Up/Down arrow keys to cycle between 0dB, +20dB, and +40dB. Press Enter to confirm and return to VU Mode.
Note: It is recommended to return this to 0dB after sampling to avoid sudden loud noises during playback.
Option 4: Threshold
Behavior: Sets a volume threshold tripwire for "Arm Sampling."
Display: Arm Slider #1.
Controls: Move the far-left vertical slider (Slider 1). The LCD VU meter will show a marker indicating the threshold level.
Option 5: Sample Length
Behavior: Defines how long the recording time will be.
Display: Length 2.5 secs Use Slider #1.
Controls: Move Slider 1 to adjust the length from 0.1 seconds up to 2.5 seconds. Press Enter to lock it in.
Option 6: Resample
Behavior: A shortcut to immediately record over the last used pad, using the same length and routing settings previously defined.
Display: Shows the pad number with an asterisk (indicating it will be overwritten).
Exit: You can abort this by pressing any other module button or running Option 2 to select a different pad.
Option 7: Arm Sampling
Behavior: Puts the SP-1200 in standby mode. It will automatically start recording as soon as incoming audio breaches the threshold set in Option 4.
Display: Sample Armed. Once audio hits, it changes to Sampling.... When finished, it reads Sample is Good (or Sample Overload if the audio clipped).
Option 9: Force Sampling
Behavior: A manual trigger. The moment you press 9 on the keypad, recording begins immediately, bypassing the threshold.
Display: Sampling... followed by Sample is Good or Sample Overload.
2. The Sync Module
To enter this mode, press the Sync button. The red LED above it will illuminate.
Option 1: Internal Clock
Behavior: The SP-1200 acts as the master clock. It will output SMPTE and Metronome/Clock signals.
Display: Select Internal.
Option 2: MIDI Clock
Behavior: The SP-1200 becomes the slave and syncs to an external MIDI clock.
Display: Select MIDI.
Note: If the external device sends clock but not start/stop messages, you must arm the Run button on the SP-1200 before starting the external device.
Option 3: SMPTE
Behavior: Sets the framerate for SMPTE synchronization (primarily used for video production or syncing tape machines).
Display: SMPTE Format is: XX fps.
Controls: Use arrow keys to select 24, 25, 30, or 30-drop frames per second. Press Enter to confirm.
Option 4: Click
Behavior: Configures the analog click output to sync with older, pre-MIDI drum machines (like LinnDrums or Oberheims).
Display: Click Divisor: XX.
Controls: Enter a division number via the keypad to divide the internal 24 PPQ (Pulses Per Quarter Note) clock. For example, a divisor of 4 sends a 48 PPQ signal.
3. The Disk Module (SD Card)
To enter this mode, press the Disk button. The red LED above it will illuminate. The Rossum uses a virtual floppy disk system stored as image files (.sp12) on an SD card.
Navigation: Use the Left/Right arrows to scroll through files. Hold down an arrow to auto-scroll. Folders are denoted by a forward slash (e.g., Folder/). The root directory exit is denoted by ../. Press Enter to open a folder.
Option 1: Save Sequences - Saves only sequence/song data to a virtual disk file.
Option 2: Save Sounds - Saves only audio samples to a virtual disk file.
Option 3: Load Sequences - Clears current sequence memory and loads sequences from a selected file.
Option 4: Load Segment # - Prompts you to enter a 2-digit number to extract a single specific sequence from a saved file.
Option 5: Load Sounds - Clears current memory and loads all sounds from a selected file.
Option 6: Load Sound # - Prompts you to select a pad, then loads a single sound from a saved file into that pad.
Option 7 & 8: Catalog Sequences / Sounds - Allows you to scroll through and view the names of sequences or sounds stored within a specific image file without loading them.
Option 9: Save All As (Rename)
Behavior: Saves all current sounds and sequences into one file.
Display: Prompts for a file name.
Controls: Use Slider 1 to cycle through letters/numbers/symbols. Use Left/Right arrows to move the cursor. The Erase button deletes characters. The Insert (Up Arrow) button adds spaces. Press Enter to save.
Option 0: Load All - Clears all memory and loads an entire disk image (sounds and sequences).
Option 27: Create Folder - Creates a new folder on the SD card. Press Yes to confirm, name it using the slider/arrows, and press Enter.
Option 28: Update Software - Used to load firmware (.bin) updates from the SD card.
4. Programming & Performance
Segment Mode
Activation: Press the Segment button. The red LED turns on.
Display: Seg XX (XX is a number from 00-99) alongside the current Tempo.
Controls: Use arrows or the keypad to select a segment.
Recording: Hold Record and press Run. Both LEDs will light up. The metronome clicks, and you can play the pads to record.
Erasing: While in Record/Run mode, hold the Erase button and hold down a pad to delete its specific hits as the playhead passes over them.
Segment Length: Press the Segment Length button, type a 2-digit number for the amount of bars, and press Enter. (Default is 02). If you shorten a segment, press Enter again to officially Truncate (permanently delete) the hidden data.
Song Mode
Activation: Press the Song button. The red LED turns on.
Display: Song XX (00-99).
Behavior: Chains segments together.
Arranging: Hold Record and press Run. Type the 2-digit Segment number you want to play first, then hit Enter. Type the next segment number, hit Enter. Press Stop when finished.
Step Program Mode
Activation: Press Step Program. The Record LED will light up, but the Run LED will not.
Display: Shows Measure, Beat, and Sub-beat (M:01 B:01 S:01).
Controls: Use arrow keys to manually jog forward/backward through the timeline. Tap or hold Run to jog continuously. Hit a pad to place a sound on the exact step. Hold Erase and hit a pad to delete a sound on that step.
Swing & Auto Correct
Swing: Press the Swing button. Use arrows to choose 50% (off), 54%, 58%, 63%, 67%, or 71%. Values with a 'T' (e.g., 67T) apply swing to triplet grids.
Auto Correct (Quantize): Press Auto Correct. Use arrows to choose note divisions (1/4, 1/8, 1/16, 1/32, Hi-Res).
Note: Both of these settings are destructive; they permanently alter the timing of notes as you record them.
Tap Repeat
Behavior: Holding this button while holding a pad will rapidly re-trigger the sound based on the current Auto Correct timing value.
5. The Setup Module (System & Editing)
To enter this mode, press the Setup button. The red LED above it will illuminate.
Option 11: Multi-Pitch
Behavior: Takes one sound and spans it across all 8 pads at different pitches to play melodies.
LED/Display: The Tune/Decay LED automatically lights up. The display shows the tuning values.
Controls: Move the 8 vertical sliders to change the pitch of each respective pad.
Exit: You must run Option 13 (Exit Multi-Mode) to return the pads to normal.
Option 12: Multi-Level
Behavior: Spans one sound across all 8 pads at different volumes (useful for ghost notes/dynamics).
LED/Display: The Mix LED automatically lights up.
Controls: Move the 8 vertical sliders to adjust the volume of each pad.
Exit: Run Option 13 (Exit Multi-Mode).
Option 14: Dynamic Buttons
Behavior: Turns pad velocity sensitivity on or off.
Controls: Press Yes or No on the keypad.
Option 17: Channel Assign
Behavior: Routes a specific pad to one of the 6 individual physical outputs on the back of the unit.
Controls: Hit a pad, enter a number (1-6) on the keypad, hit Enter.
Note: Because outputs are monophonic, routing two sounds to the same channel creates a "mute group" (e.g., putting an open hi-hat and closed hi-hat on Channel 1 will cause the closed hat to cut off the open hat).
Option 18: Decay / Tune Select
Behavior: Toggles the function of a pad's vertical slider. By default, the slider changes Pitch (Tune). This option changes it to control the volume Envelope (Decay).
Display: Changes to show Decayed.
Controls: Select a pad, hit 2 on the keypad. The slider will now make the sound shorter/choppier. To revert to pitch control, run Option 18 again and press 1.
Option 22: Dynamic Allocation
Behavior: By default, if you rapidly hit a long sample (like a crash cymbal), the sound cuts itself off to replay. Turning this to Yes allows the sample to bleed over into a second internal voice channel so the tail rings out naturally while the new hit starts.
Option 25: Reverse Sound
Behavior: Plays a sample backward.
Controls: Select a pad, press Yes on the keypad to confirm.
